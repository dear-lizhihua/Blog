转码工具-货币转换

June 5, 2021 · One min read
lizhihua
大众光明导航负责人

介绍#

实现一个简单的 转码工具

知识要点#

  1. 除了使用 flex layout，首次尝试使用 grid layout。
  2. 对货币工具建立模型层、服务层，减少视图层交互，与 Vue API 解耦。
  3. 将模型层实例对象转为响应式对象，承载业务逻辑和响应式数据。
  4. 将配置过的 axios 实例，作为 api client 注入到服务层，避免多个项目重复配置。
