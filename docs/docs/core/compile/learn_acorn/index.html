简单理解 acorn

参考链接#

  • www.npmjs.com/package/acorn
  • github.com/acornjs/acorn
  • webpack 与 rollup 背后的 acorn

设计意图#

acorn 是一个 JavaScript parser。

我的理解：

  1. 用于帮助用户从字符串形式的源语法（例如 ES2015 语法）中，解析出抽象语法树(abstract syntax tree)。
  2. 用户在遍历抽象语法树的过程中，修改或替换抽象语法树的节点(node)，再转为字符串形式的目标语法（例如 ES5 语法）。

从字符串得到抽象语法树#

parse(input, options)#

parse 方法是 acorn 的一个主要接口。输入参数是一个字符串，返回值将是 ESTree 规范 规定的抽象语法树对象。

从 string 解析出 AST object#

const acorn = require('acorn')const ast = acorn.parse(`import $ from 'jquery';`, {  sourceType: 'module',  ecmaVersion: 6})

得到抽象语法树的 JSON string 形式#

通过 JSON.stringify() 得到 ast object 的 JSON string 形式#

console.log(JSON.stringify(ast, null, 2));
{  "type": "Program",  "start": 0,  "end": 23,  "body": [    {      "type": "ImportDeclaration",      "start": 0,      "end": 23,      "specifiers": [        {          "type": "ImportDefaultSpecifier",          "start": 7,          "end": 8,          "local": {            "type": "Identifier",            "start": 7,            "end": 8,            "name": "$"          }        }      ],      "source": {        "type": "Literal",        "start": 14,        "end": 22,        "value": "jquery",        "raw": "'jquery'"      }    }  ],  "sourceType": "module"}

AST explorer 提供了方便的 UI 界面，同样可以得到 ast object 的 JSON string 形式

遍历抽象语法树#

常见的遍历接口设计#

  1. 在 estraverse 中，我们可以进入节点和离开节点，也可以跳过节点、替换删除、中断遍历、替换节点
// 进入节点和离开节点estraverse.traverse(ast, {  enter: function (node, parent) {    if (node.type == 'FunctionExpression' || node.type == 'FunctionDeclaration')      return estraverse.VisitorOption.Skip;  },  leave: function (node, parent) {    if (node.type == 'VariableDeclarator')      console.log(node.id.name);  }});// 跳过节点、替换删除、中断遍历、替换节点estraverse.traverse(ast, {  enter: function (node) {    this.skip();    // this.remove();    // this.break();    // if (node.type === 'Literal') return replaced;  }});
  1. 在 @babel/traverse 中，我们可以进入节点和进入特定类型的节点
// 进入节点traverse(ast, {  enter(path) {    if (path.isIdentifier({ name: "n" })) {      path.node.name = "x";    }  },});// 进入特定类型的节点traverse(ast, {  FunctionDeclaration: function(path) {    path.node.id.name = "x";  },});
