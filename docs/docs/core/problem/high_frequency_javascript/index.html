实现一个异步串行？#

例如：promise1 是列表接口，promise2 是详情接口，promise3 是详情接口返回的某个字段的关联接口。

伪代码如下：

(promise1 execute => promise1 ok) =>(promise2 execute => promise2 ok) =>(promise3 execute => promise3 ok) =>callback

实现一个异步并行#

伪代码如下：

(promise1 execute => promise2 execute => promise3 execute) => // 执行顺序固定或不固定都可以(promise3 ok => promise1 ok => promise2 ok) => // 返回顺序不固定callback

如何解决 promise 回调地狱#

promise1.then(function() {  promise2.then(function() {    promise3.then(function() {      callback()    })  })})
