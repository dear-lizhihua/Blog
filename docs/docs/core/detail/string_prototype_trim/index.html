实现一个 String.prototype.trim() 方法

常规实现#

正则示意图

String.prototype.trim = function() {  return this.replace(/^\s+|\s+$/g, ''); // g:全局}

MDN 实现#

正则示意图

String.prototype.trim = function () {  return this.replace(/^[\s\uFEFF\xA0]+|[\s\uFEFF\xA0]+$/g, '');};

core-js@3 实现#

正则示意图

core-js@3 源码

  1. https://github.com/zloirock/core-js/blob/master/packages/core-js/internals/string-trim.js
  2. https://github.com/zloirock/core-js/blob/master/packages/core-js/internals/whitespaces.js

从其源码我们可以看出，core-js@3 替换了以下空白符和这些空白符的全部组合：

  • \u0009
  • \u000A
  • \u000B
  • \u000C
  • \u000D
  • \u0020
  • \u00A0
  • \u1680
  • \u2000
  • \u2001
  • \u2002
  • \u2003
  • \u2004
  • \u2005
  • \u2006
  • \u2007
  • \u2008
  • \u2009
  • \u200A
  • \u202F
  • \u205F
  • \u3000
  • \u2028
  • \u2029
  • \uFEFF

简化后代码如下

var whitespaces = '\u0009\u000A\u000B\u000C\u000D\u0020\u00A0\u1680\u2000\u2001\u2002' +'\u2003\u2004\u2005\u2006\u2007\u2008\u2009\u200A\u202F\u205F\u3000\u2028\u2029\uFEFF';var whitespace = '[' + whitespaces + ']';var ltrim = RegExp('^' + whitespace + whitespace + '*');var rtrim = RegExp(whitespace + whitespace + '*$');String.prototype.trim = function () {  var string = this;  string = string.replace(ltrim, '');  string = string.replace(rtrim, '');  return string;};
