测试 HTML, CSS, JavaScript 大小写敏感

定义#

  • 不识别：指大小写不敏感，例如 test 和 TEST，都被认为是 test。
  • 识别：指大小写敏感，例如 test 和 TEST，被认为是两个不同的值。

HTML：属性名不识别，属性值识别#

  1. 属性名 ID 和 Id 都会被认为是 id
  2. 属性值 ID 和 Id 查询结果是不同的 div
  3. 属性值 id 查询结果 null
<div ID="ID"></div><div Id="Id"></div><script>  console.log(document.querySelector('#ID')) // <div ID="ID"></div>  console.log(document.querySelector('#Id')) // <div Id="Id"></div>  console.log(document.querySelector('#id')) // null</script>

CSS：属性值识别#

  1. class 类名识别。属性值 ID 和 Id 选中的是不同的 div
<style>  .id { /* 无法选中任何元素 */    width: 100px;    height: 100px;    border: 1px solid red;  }  .ID { /* 可以选中 id=ID 的元素 */    width: 100px;    height: 100px;    border: 1px solid green;  }  .Id { /* 可以选中 id=Id 的元素 */    width: 100px;    height: 100px;    border: 1px solid blue;  }</style><div CLASS="ID"></div><!-- 绿色 border --><div class="Id"></div><!-- 蓝色 border -->
  1. style 属性名不识别。
  2. style 属性值不识别。
<div style="BORDER: 1px solid green; width: 100px; height: 100px;"></div><!-- 绿色 border --><div style="border: 1px solid BLUE; width: 100px; height: 100px;"></div><!-- 蓝色 border -->

JavaScript：变量名、字符串值、对象属性名都识别#

<script>  var ID = 'ID'  var Id = 'Id'    console.log(ID) // ID  console.log(Id) // Id  console.log('ID' === 'Id') // false    var obj = {    ID: 'ID',    Id: 'Id'  }  console.log(obj.ID) // ID  console.log(obj.Id) // Id</script>

结论#

-属性名或属性值识别
HTML属性名不识别
HTML属性值识别
CSS类名识别
CSS属性名不识别
CSS属性值不识别
JavaScript变量名识别
JavaScript字符串值识别
JavaScript对象属性名识别
