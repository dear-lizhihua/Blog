使用 Vite 构建一个集成 ant design 的 Vue 3 应用程序

参考资料#

  • Ant Design of Vue。

初始化项目#

我们先按照 vite 文档 初始化项目。

$ npm init @vitejs/app$ yarn create @vitejs/app // 或者 yarn
  • Project name: @lizhihua/app
  • Select a framework: vue
  • Select a variant: JavaScript

生成项目的特性#

// Vue project - package.json

{  "name": "@lizhihua/app",  "version": "1.0.0",  "scripts": {    "dev": "vite",    "build": "vite build",    "serve": "vite preview"  },  "dependencies": {    "vue": "^3.0.5"  },  "devDependencies": {    "@vitejs/plugin-vue": "^1.2.2",    "@vue/compiler-sfc": "^3.0.5",    "vite": "^2.3.0"  }}

通过 Semantic Versioning ，我们注意到生成项目的一些特点：

  • scripts 中除了 dev 和 build，vite 还额外提供 serve。serve script 对应的 vite preview 命令，可以实现 部署静态站点 的能力。我们可以通过运行 yarn run build 得到 dist 构建目录，然后运行 yarn run serve 将 dist 文件夹的资源部署到本地服务中。这时候按照终端的提示，直接访问 http://localhost:5000 来 serve 整个 dist 目录下的静态资源。
  • Vue 的 MAJOR 版本是 3
  • vite 的 MAJOR 版本是 2
  • @vue/compiler-sfc 版本与 Vue 版本保持一致。在 Vue@2 中，也是 vue-template-compiler 版本与 Vue 版本保持一致。
  • @vitejs/plugin-vue 插件来编译 vue 组件。vite 不单支持 vue，同时还支持 许多其他模板，例如 react 项目就是使用 @vitejs/plugin-react-refresh 插件来编译 react 组件

// React project - package.json

{  "dependencies": {    "react": "^17.0.0",    "react-dom": "^17.0.0"  },  "devDependencies": {    "@vitejs/plugin-react-refresh": "^1.3.1",    "vite": "^2.3.0"  }}

安装依赖和运行项目#

$ cd @lizhihua/app$ yarn$ yarn run dev

安装 vue-router@4#

$ yarn add vue-router@next

我们按照 vue-router@4 文档 配置路由

改造 main.js 文件

import { createApp } from 'vue'import App from './App.vue'import { router } from './router'
const app = createApp(App)app.use(router)app.mount('#app')

改造 App.vue，配置 router-link 和 router-view

<template>  <h1>Hello App!</h1>  <p>    <router-link to="/">Go to Home</router-link> |    <router-link to="/about">Go to About</router-link>  </p>  <router-view></router-view></template>

创建 Home 页面

<template>  <h1>Home</h1>  <div>{{state.text}}</div></template><script setup>import { reactive } from 'vue'const state = reactive({ text: 'this is home page' })</script>

创建 About 页面

<template>  <h1>About</h1>  <div>{{state.text}}</div></template><script setup>import { reactive } from 'vue'const state = reactive({ text: 'this is about page' })</script>

创建 router/index.js 文件

import { createRouter, createWebHistory } from 'vue-router'import Home from '../pages/home/index.vue'import About from '../pages/about/index.vue'
const routes = [  { path: '/', component: Home },  { path: '/about', component: About },]
export const router = createRouter({  history: createWebHistory(), // createWebHistory | createWebHashHistory  routes})

安装 vuex@3（可选）#

$ yarn add vuex@next

已经很多年没用过 vuex，这里省略配置

安装 sass#

$ yarn add -D sass

安装后，向 home 页面添加 scss 样式来进行测试

<style lang="scss" scoped>h1 {  color: #42b983;}</style>

安装 Antd Vue#

安装依赖#

$ yarn add ant-design-vue@next // 注意使用 @next 安装 ant-design-vue@2。官网文档写错了，会安装 ant-design-vue@1$ yarn add @ant-design/colors // ant-design 配色

使用 Antd Vue#

在 main.js 增加如下

import { DatePicker } from "ant-design-vue";import "ant-design-vue/dist/antd.css";app.use(DatePicker);

测试效果#

在 antd vue 找个示例

<template>  ... 在 template 最后位置增加 date-picker 组件  <a-date-picker v-model:value="value1" /></template><script setup>import { ref } from 'vue'...const value1 = ref() // 在 script 最后位置增加 value1，它的值类型是 ref。</script>
