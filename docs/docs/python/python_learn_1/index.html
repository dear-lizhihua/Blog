字符串、变量

Syntax 语法#

print()#

和 JavaScript 中的console.log() 类似

print('hello world')

字符串#

使用 单引号 和 双引号 将 字符 括起来，这点和 JavaScript 相同。

后面示例为了省事（可以少按 shift），我都会使用单引号。大家还是多找些好的编码规范。

// main.py'hello world' "hello world"

字符串插值#

JavaScript 里面是使用 模板字符串 实现字符串插值。

Python 使用两个要素实现：f 函数 和 {} 括号

这里预先用到了声明变量，不过理解上应该不是问题。

first_name = 'zhihua'last_name = 'li'fullname = f'{first_name} {last_name}'print(fullname)

声明变量#

和 JavaScript 不同，Python 不需要 var, let, const 等声明关键字

message = 'hello world'print(message)

API 接口#

title 函数#

单词首字母大写，类似 JavaScript 中的 lodash.capital()。

print('ada lovelace'.title()) // Ada Lovelace

upper 函数#

print('Ada Lovelace'.upper()) // ADA LOVELACE

lower 函数#

print('Ada Lovelace'.lower()) // ada lovelace

strip, rstrip, lstrip 函数#

查了文档

  • r 表示 right side（str.rstrip 没说，这篇有提到 Python Trim String – rstrip(), lstrip(), strip() 2 Comments）
  • l 表示 left side（str.lstrip 说是 leading characters。我觉得左右的说法更通顺些）
print(' python '.strip()) // 'python'print(' python '.rstrip()) // ' python'print(' python '.lstrip()) // 'python '
