缘起#

重要声明

  1. 完全是个人观点，看法可能不够全面，此文仅用于梳理
  2. 并非针对任何国家，单纯以自身过往经历，从人性和组织的角度，观察整个资本主义异化后的社会关系。

此文待展开，并逐步更新。我会逐步把自己所认识的中国商业形态，以及组织和个人的在发展中遇到的问题，以理性的语言进行描述。

资本家代表：马云。没有大家认为的那么高的价值#

重要声明

马云是一个资本主义异化的代表，他可以代表一些资本家的心态。

不可否认，马云前期作为是商业的润滑剂，他也有着可以称道的商业贡献。

但从初高中基本政治教育进行反思，从中国整个社会生态来看，他的价值没有那么高。

马云存在的认知问题#

  1. 以理想主义号召，实则伪善。嘴上都是主义，心里都是生意。
  2. 公开宣传 996 违法。并且为违法披上商业道德的外衣。从本质上看：
    1. 毋庸置疑 996 本身违法。我觉得要么国家修改劳动法延长法定工作时间，要么依法办事拿下这些资本家。否则无有为 996 进行辩解的余地。劳动仲裁委员会无视集体加班现象，这点个人不做评论，现阶段(2021.04.18)不喜欢加班的自己想办法，没事多去脉脉上面喷。
  3. 制造的商业利益被少数人持有，无法与大众共享。马云阶层喜欢创造商业神话，鼓吹制造财富自由。从本质上看：
    1. 鼓吹财务自由的白日梦。2021 年胡润财富对自由门槛定义，中国一线城市入门级财富自由门槛是 1900 万元，中级要 6500 万，高级要 1.9 亿。人人都财务自由怎么可能实现？此等谎言是引诱作为社会中坚力量的优秀青年为其阶层服务，只会制造更为广泛的奴役现象。如果优秀青年都信仰此等理念，只怕中国底层社会离卖儿卖女、水深火热、女性无地位的旧制度不远了。
    2. 上市套现过程对于基础科研并无实益，科研人员把精力放在短期可见的商业利益，不重视基础研发，一旦上市财务自由，这个国家最为优秀的人才就开始放弃奋斗，甘于享乐，朋友圈炫富之风不绝于耳。长期对于整个社会科技发展越不利。
    3. 不断制造新的阶层，为接受过高等教育的知识分子提供利益，维系其上层社会关系。这些人中的多数每天辛苦工作，这些人的少数人靠压榨多数人。制造阶级即制造奴役，在利益诱惑下，知识分子、工人中的一些人，为了蝇头小利可以出卖自身阶层，心甘情愿成为资本家的走狗。
    4. 装饰道德。资本家掌握巨大的利益，这时付出一点些微的好处，就可以博得一片喝彩，这让广大兢兢业业的劳动者寒心。
  4. 垄断话语权。新闻媒体逐步被这些人把控，到处宣扬资本主义的糖衣炮弹。
  5. 资本异化现象逐步严重。附：资本异化话题有待以后展开。
  6. 阻碍社会发展。
    1. 阻碍经济发展。加班造成最能消费的群体为银行卡上的数字打工，缺少时间和精力进行消费。
    2. 阻碍社会进步。在共产主义下，商业组织又重新恢复回奴隶制，任由时间支配、身体健康和精神健康等权利交给资本家控制，这实则是社会倒退。不过我觉得社会就是在倒退和进步之间不断：当民众厌恶压迫，反抗被施加的恶行，社会关系就会比较进步；当民众无视压迫，接受被施加的恶行，社会关系就会变得倒退。
    3. 垄断资源。把控诸多社会资源，却缺乏有效的道德约束。商人奉行利益至上，无视诸多道德，这点马克思早就说过，如果有 300% 的利润，资本就敢于践踏人间一切法律。附：组织之恶话题有待以后展开。

更美好的社会#

你我之辈，忍将夙愿，付与东流？——传毛泽东所写，真伪暂且不辨

  1. 中国有九亿农民，养活了其他几亿人。体能上要辛苦种地，享受上生活并不富足，社会上也不受尊重。
  2. 6 亿人每月收入在 1000 元以下。普通青年在大城市有了基本的生活保障后可以适当享受，但为追求刺激过度消费而被控制就很不值得，更无需渴望所谓财务自由的空想。

一个国家要实现公平正义，那就要更多响应党的号召，为人民服务，为创建更美好的社会的理想服务。

有为青年朋友要多关注更大范围内的底层人民，而非成天盯着这些资本家的伟光正。

努力为一个不靠出卖灵魂的人民，为实现精神层面开拓、自由、快乐而奋斗。

为生活奔波的朋友，能做到的也可以是快乐的生活，周末活动多发朋友圈，气死那些周末还在工作的资本家，让他们为财务自由去忙碌吧。

结论：#

  1. 马云代表商业社会的头部阶层，无有道德约束，奉行垄断独享一切利益。
  2. 马云是一个披着理想主义外衣，实则精致利己主义的商人。
  3. 马云是一个无视基层员工利益，强行以商业上层视角展开其企业文化，严重脱离人民群众。
  4. 马云制造身体和精神两方面和压迫，对劳动者的时间进行残酷的剥削。
  5. 马云公开宣传违法行为，突破中国社会基本道德底线，属于高级人渣。
  6. 信仰马云的时代过去了，无产阶级人民终将认清其虚伪面貌，站在党和共产主义的一边。
