大众光明成立了

缘起#

从 2021.04.21 起决定退出印记中文，04.21-04.26 一直在筹备新的组织，大众光明 因此而迅速成立了，大众光明寓意引导大众走向光明。本来想使用人民光明，但是听着不够大气。其实我们使用了德文的 volks，正有人民的含义。

说说我的想法#

  1. 现在方向有点偏。
    1. 文档缺乏有效推进。比如我们搞公众号、搞周刊，实际收益并不高，而重要的文档因为缺乏有责任、能力强的成员维护，反而质量越来越差。在张老师讲公开课时，听到提到没有好的 Rollup 中文文档，我感觉深受触动，我们做这么多年文档，除了 webpack 是我一直推进，React 社区成员众多，Vue, Node 这些文档都是私人维护，其他哪一个是我们做成的呢？而众多重要文档 lodash, Babel 喊了很久，没有一个做成的，也有做到一半就半途而废的。
    2. 理想缺失，功利主义泛滥。大家都在为公众号用户达到一万两万，为公众号热点文章有 1000 阅读量而庆幸，转发些其他公众号的推广，现状着实令人感到羞愧。作为一个深入做文档的组织，背离了初心忘记了自己的使命，理想的丧失着实令人感到痛心。
  2. 开源化的文档不可行。
    1. 无法把控翻译质量，关注虽多但持续贡献高质量的人不多。
    2. 机器翻译越来越强大。我们应该更多探索机翻+人工校对+持续集成方面的投入，但这方面的进展缓慢，大家只是嘴上说支持，但并未见实际行动。
  3. 做的一些事没有精准把控用户需求，好像在隔靴挠痒，自己的专业都没做好就去搞其他的。
    1. 导航覆盖范围仅限于前端。
    2. 站内文档搜索功能羸弱。
  4. 开源社区极其低效。
    1. 由于不是公司组织形式，团队缺少配合，开始互相指责。
    2. 缺乏快速产出原型的能力，好的想法得不到有效实践。

不得不承认，这些年的实践其实是失败了，幻想中的利益人民群众其实是成就出一些精致利己主义者。

我想做个纯粹点的社区#

谈谈组织管理和心态

  1. 减少精致利己主义者的加入。
    1. 那些精致利己的技术再好，也不会和你共享，只有依赖自身，依赖组织内部的团结一心，才有力量。
    2. 依赖身边信得过朋友：比如我的同学，他在后端方面基础扎实技术精湛，现在大众光明的后端都是他在维护；比如支持我的老银，他一直想参与文档的建设，并且以前给我提过很多很有用的人生经验，可惜我充满幻想没听进去。还有一些社区的朋友，真正能看出他们发自内心的对于我们从事事业的喜爱。想到这些，我就觉得一定不能辜负他们的期望。
  2. 强大自身。自己要向全栈发展，减少对于他人的依赖。现在在一家从事云服务的小公司，二三十人也没许多事，我的直属领导给我一台虚拟机实例，最近在谷歌云上面各种搞，对于 Google Cloud Run, Google Compute Engine, gcloud, Docker, SSH, shell 方面很有收获，后端大哥技术实力也很强，领导还让我学 Python，这段时间可以总结几篇 Python 文章分享出来。总之，难者不会，会者不难，更相信自己的潜力，而不是寄希望于他人。
  3. 广泛听取组织讨论。一旦确认方案就要大家团结起来全力以赴地实践，避免推诿扯皮互相指责。

谈谈推进文档建设

  1. 基本原则要按部就班的推进，态度端正避免心浮气躁，更不要捡了芝麻丢了西瓜。webpack, Rollup, Babel, lodash, koa 这些文档我们需要挨个捡起来，我准备先从 Babel 开始。
  2. 推进基于 CI/CD 的机器翻译。
    1. 借助强大的 Google 翻译或者 DeepL 翻译加快文档翻译速度。不过机器翻译只能辅助，并不能真正的解决文档的核心问题，得先掌握背后的技术，才能把文档翻译水平搞得精湛，把文档翻译质量提高上去。所以这方面我也得多花点时间，把上面提到的文档的技术掌握。
    2. CI/CD 方面我还欠缺经验，需要多投入时间精力掌握背后 Linux 体系的一套工具，提高服务的运维效率。最近也是把 shell, scp, SSH 掌握了，能写个一键部署脚本，简化不少繁琐地构建和部署工作。
  3. 配套文档我们还需要建立导航站点，有了导航站点，用户就不会迷路。为此我想了很久，终于在 2021.04.26 开发出第二版 大众光明导航。

我的期望是要成为一个有理想、有信仰的组织，顺便再谈谈一些主观感受#

  1. 为混老板的年薪百万而刷 KPI，名为开源实为精致利己，嘴上都是主义，心里都是生意。用这种态度做开源项目，充满着傲慢，反正你们爱用不用。难道挣钱的事就好好做，利益广大开发者的事就敷衍了事。大公司占着大量的社会资源，只精心产出高质量的商业产品，中层管理为讨好老板和老板的劳斯莱斯费尽心机，这种麻木不仁的现状太可怕了。
  2. 现在朋友圈炫富风极其令人反感，社会上没有信仰、没有正义。
    1. 我觉得我们就是要承担起责任正本清源，要树立好的正确的形象，为好的中文社区而奋斗。
    2. 不要整个几百万就天天玩港股美股，背离完工农阶层，靠资本运作剥削民众。社会上创造那么多财富，如果不与人民群众共同享用，那就只会是让社会底层更加艰难数字而已。人不过一日三餐一床而眠，有钱了就踏踏实实过日子，更要想想怎么服务大家。
  3. 大家要有担当，做事有底线。
    1. 就事论事，不能谁强就向着谁。阿里势力强大，我们连说他们 cnpm 部署做的不好都要注意影响，这不很可笑。还有人拿 cnpm 使用人数众多帮到过很多人来反驳我，这我难道就不知道，一码归一码。也可能是我太菜，花了两天时间，愣是没把 node 和 docker 两种方式搞定，app 怎么都连不上数据库，github issues 也搜了，各种方法也试了。最后只好使用 Nexus 3 一键部署搞定，反正公司小，也就我一人用这个发布 package。另外 yapi 的官方部署也很烂，不过有好用的第三方部署方案 fjc0k/docker-yapi，也算是过关了。其实我以前也说过 ant design 好用，否则也不会在导航上创建 ant design 系列和 umi 系列。当然这些都是我的主观开发和亲自实践，接受各位的批评。
    2. 不对的就认真喷，而不是明哲保身。你们不觉得电视和民众上面把一个公开宣扬违法行为(996 是福报)的分子当偶像很可怕吗？大家都把宋江、李逵当成忠义，那社会主义社会和古代封建社会又有何区别。
    3. 先代表自己。不能被身份限制，自己要是个阿里、百度的，就不能喷马云、李彦宏了，如果不为人民说话、做了恶行，那就照样说他。
    4. 身为红色政权下的人民群众，对于资本家要有底气。给大家一个真正的示范，告诉所有人，离开大资本照样可以混得好。

我们是一个有理想有信仰的组织，不能随波逐流，否则何以对得起这些对我们寄予厚望的志同道合者。
