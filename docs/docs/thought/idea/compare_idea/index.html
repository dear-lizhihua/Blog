几种思想的比较：古希腊哲学、古印度佛学、现代中国共产主义

重要声明#

这篇用于梳理，都是个人看法。

2021.05.23

先解决有无的问题，整理一部分内容，以后再补充更为细节内容。

古希腊

古希腊哲学#

奴隶制#

古希腊依赖战争掠夺他国民众，畜养奴隶提供生产力，来减少本国青年男子的劳动，他们可以脱离生产，有大量时间和精力接触艺术、文学、喜剧、军事、哲学、数学等抽象领域。

多元化的有神论#

古希腊人是一个宗教信仰强烈的民族，他们供奉许多的男神与女神。

古希腊宗教虽然没有具体的宗教典籍，但中心教义是：认为大自然的山川、四季、植物、星辰、日子、湖泊、海洋、天空、大地乃至命运、技艺、七情六欲都有神祇在掌管。不过古希腊宗教的世俗性质很重，因此人们对神祇都以尊敬且祈求平安的心态来供奉神祇。

民主#

人无事可做就会产生各种思想，想的多了产生诸多意见的分歧，但重大决策终归是要有一个共识的。民主由此产生，基于投票贯彻多数人的看法。

现代社会更像是古希腊社会的延续#

我们很容易天真地以为，古代文明的艺术创造是辉煌的。殊不知，人类的文明就是一部剥削和反抗剥削的血泪史。在中国古代社会，十分之九的农民阶层被皇权剥削和压迫，另外十分之一的地主阶层负责创造文明。

我认为，现代社会更像是古希腊社会的延续，我们吃的、住的、穿的、用的，都是由十之七八的农民、工人的生产。只不过

  • 化学肥料的广泛使用，缓解了口粮上的矛盾。
  • 电力、生产机械的广泛使用，改善了工人的劳动强度。
  • 共产主义制度的出现，为工农阶层提供了基础保障。

理想#

古希腊出现了一位大哲学家苏格拉底，他的弟子柏拉图总结了他的思想，我们通过柏拉图的书籍，知道了他的想法和做法。

学术传承#

在军事上，马其顿的亚历山大三世征服了古波斯、古印度地区。因此这些地区受到古希腊学术的影响。

古罗马深受古希腊的影响，在分裂后产生东罗马拜占廷帝国、西罗马帝国，在极简欧洲史这本书中，我们了解到，古罗马教会传抄古希腊的典籍。

从维基百科的介绍中，我们了解到：

在欧洲的修道院图书馆中和没落的拜占庭帝国里，文艺复兴时期的思想家们搜集到了古典时代的文学、历史和演说文献。这些文献大多使用拉丁语或古希腊语写成，多数内容晦涩难懂。文艺复兴时期的学者同12世纪文艺复兴时期的中世纪学者最显著的差别在于，前者将文学和历史等文化方面的文献作为新的研究重点，而后者只专注于自然科学、物理学和数学的希腊语和阿拉伯语典籍。

文艺复兴时期学者的一大成就，就是自古典时代后期以来，第一次将希腊语文化作品从整体上引入了西欧。该运动让希腊语的文学、历史、演讲和神学资料重新进入了西欧学校的课程中。

  • 文艺复兴时期的思想家们从没落的拜占庭帝国尘封的故纸堆里，找出了拉丁语或古希腊语书写的古希腊文化，并且重新学习其中的思想。

古印度佛学#

觉悟之学#

佛，又称佛陀，英文 buddha，意译为“悟道者”。佛学，即是觉悟之学。

古印度的自然环境#

古印度气候炎热、雨水充沛，夏季有着长达三个月的雨季。所以古印度人每年夏季都需要长期闲居。

古印度的社会环境#

在这种天然不需要过多生产力，就能获得口粮，那思想领域的果实更是层出不穷。

文化冲突#

语言不统一#

辩论真理#

释加牟尼佛证道后，通过与异教徒展开辩论，来收服观点不同、按照其他理念修行的沙门。

众生和教徒#

佛教的教育受众是众生。

佛教的组织形态是教徒，从道德的层面来看，他们各个都是道德标兵（不近女色、极低的欲望、远离是非、奉行十善、善于思考、勤于辩论）。

平等理念的产生#

古印度有四种阶级：

  • 婆罗门。负责祭祀，享有诸多特权。可以理解为皇权阶层
  • 刹帝利。负责政治和军事，保护婆罗门。可以理解为官僚阶层
  • 吠舍。农民或牧民，生产食物，可以经商。可以理解为工农商阶层
  • 首陀罗。没有人身自由的奴仆，提供各种服务。可以理解为奴隶阶层

佛教的理念是一切众生平等，既然大范围的六道众生都平等了，那其中小范围的人类众生的平等也不用刻意提醒。

完美的众生#

佛在 大方廣佛華嚴經 如來出現品第三十七之二 中提到

復次，佛子！如來智慧無處不至。何以故？無一眾生而不具有如來智慧，但以妄想顛倒執著而不證得；若離妄想，一切智、自然智、無礙智則得現前。

不完美的世界#

佛教的世界观，此世界是娑婆世界。

娑婆世界，指释迦牟尼佛所教化的三千大千世界。此世界众生安忍于十恶，忍受诸烦恼，不肯出离，故名为忍。

戒律#

十善业是佛学中最普遍、最基础的戒律，出自具有国际主义援助精神的唐朝僧人实叉难陀，其翻译的十善业道经介绍了十善业的细节。

  • 不杀生
  • 不偷盗
  • 不邪淫
  • 不妄语
  • 不两舌
  • 不恶口
  • 不绮语
  • 不贪欲
  • 不瞋恚
  • 不邪见

唯实和心理学#

佛教传到汉唐之间，诸多古印度僧人的论述，已经发展到心理学层面。

瑜伽师地论就是本人敬仰的玄奘大师，从古印度取回并组织翻译的经文。

无神论#

佛学中没有宗教和神，神也是众生的一种。

核心理念：三世因果六道轮回#

佛学中最容易忽略的一点是，佛教的一切理念，建立在三世因果六道轮回。如果忽略了这一点，一切讨论都没有意义。

  • 三世因果：佛教认为因果法则是天然存在的，不会因为人认为不存在就不作用于此人。业为因，报为果，因和果辗转相生，谓之“因果报应”或“因缘果报”。因果通于过去、现在和未来，谓之“三世因果”。
  • 六道：三善道为天、阿修罗、人； 三恶道为畜生、饿鬼、地狱。
  • 轮回：一种思想理论，认为生命会以不同的面貌和形式，不断经历生死。

我们假想真的存在三世因果六道轮回，教徒们的一举一动是不是会更加小心谨慎。

学术传承#

佛学在中国的异化：救度#

传至中国本土，产生了救度的理念。在讲谈社所编写的中国的历史套装中，提到唐朝广泛流行的救度理念。

这点以后补充。

缺点#

  • 释加牟尼的道德和戒律，让普通人心生敬畏，敬则远之。
  • 难以在广泛的社会群体中展开。
    • 在古希腊、古印度那个区域，人口规模也就是一两千万人。
    • 在古代中国范围从唐至清，人口规模已经突破两千万人，普遍是五千万到一亿人口，这么多人口，以什么理论展开实践。
  • 普遍的教育受众。六道众生这个范围太大了，释迦摩尼佛陀死后无人能引导六道众生走向觉悟，下一位弥勒佛陀降世在几十亿年以后。众生又刚强难化。所以佛教具有正法、像法、末法的时代划分，越往后佛教的影响力越弱。
  • 普遍的平等性。六道众生都平等，这个空前绝后想法难以广泛展开实践。现代人的观念中，人和狗怎么能平等呢，畜生就是畜生而已，最多不过是当成宠物来养，也无法脱离主仆的关系。
  • 人性难以管控。宣扬佛法的佛教徒自身也有七情六欲，如果某位高僧大德在这方面出现问题，那对于现代社会本来就偏向科学的普通教徒，在信仰层面极容易产生动摇。

共产主义

现代中国共产主义#

共产主义是实践的学问，依赖制度和基层组织，在广泛的人群中展开思想改造。

共产主义出现在中国的有利条件#

  • 口粮这个中国社会亘古难题的缓解。
  • 人口基数大。能让这么多人民信服的理念，而且经过验证，可以算是现阶段的真理了。
  • 学术体系传承。古代中国人有留存典籍的习惯，现在我们还能了解到周朝的制度和周公的文化。
  • 没有得天独厚的自然条件，中国人自古信仰自身的力量实践。中国自古灾难频发，中国神话中有大禹九年治水，而不是像挪亚建造方舟躲避大洪水。

古希腊人、古印度人秀完他们的，这回轮到现代中国人开始思考了，而且这次还特有着理念的实践。
