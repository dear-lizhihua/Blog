在 Node.js 中以 ES modules 方式加载 js 文件

Node.js 支持 ES modules 方式加载 js 文件#

  • Announcing core Node.js support for ECMAScript modules
  • Node.js package.json field definitions

通过 package.json 的 type 字段，我们可以在 Node.js 中以 ES modules 方式加载 js 文件。

// package.json{  "type": "module"}
// index.jsexport const add = (x, y) => {  return x + y}

执行 node index.js

如何使用 __dirname 和 __filename#

Node.js 文档中提到，CommonJS 变量在 ES modules 中无法使用。所以 ES modules 中 没有 __filename or __dirname。

可以通过 import.meta.url 复制 __filename 和 __dirname 的用法。

  • path.basename(path[, ext])
  • path.dirname(path)
  • path.extname(path)
  • url.fileURLToPath(url)
import { dirname, basename, extname } from 'path';import { fileURLToPath } from 'url';
console.log(1, fileURLToPath(import.meta.url))console.log(2, dirname(fileURLToPath(import.meta.url))) // 获取路径的目录名称。等于 CommonJS 下的 __dirnameconsole.log(3, basename(fileURLToPath(import.meta.url))) // 获取路径的基础名称。等于 CommonJS 下的 __filenameconsole.log(4, extname(fileURLToPath(import.meta.url))) // 获取路径的后缀名称。
