macOS 安装 Rime 输入法

重要声明#

主要参考：

  1. macOS 安装配置 Rime

安装输入法#

通过 brew 安装（ brew install --cask 已经取代 brew cask install）

$ brew install --cask squirrel

安装 Rime 的配置管理工具 plum#

# 进入 Rime 配置目录$ cd ~/Library/Rime/# Git 获取 plum$ git clone https://github.com/rime/plum.git

使用郑码#

安装郑码输入方案 lotem/rime-zhengma#

# 进入 plum 目录$ cd ~/Library/Rime/plum$ bash rime-install lotem/rime-zhengma

启用郑码#

$ cd ~/Library/Rime$ vi default.custom.yaml // 创建 default.custom.yaml 文件，录入如下配置，注意 yaml 空格
patch:  schema_list:    - schema: zhengma

这里的 schema: 后面的名字，对应的是 ~/Library/Rime 文件 zhengma.dict.yaml 和 zhengma.schema.yaml 的名称

使用拼音#

安装地球拼音输入方案 rime/rime-terra-pinyin#

# 进入 plum 目录$ cd ~/Library/Rime/plum$ bash rime-install rime/rime-terra-pinyin

启用地球拼音#

$ cd ~/Library/Rime$ vi default.custom.yaml // 创建 default.custom.yaml 文件，录入如下配置，注意 yaml 空格
patch:  schema_list:    - schema: terra_pinyin

这里的 schema: 后面的名字，对应的是 ~/Library/Rime 文件 terra_pinyin.dict.yaml 和 terra_pinyin.schema.yaml 的名称

部署输入方案#

切换输入方案#

  1. 先在输入法之间切换，切换到 Squirrel 输入法
  2. 再使用 Ctrl + ` 在输入方案之间切换
