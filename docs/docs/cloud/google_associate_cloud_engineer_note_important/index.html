谷歌助理云工程师(ACE)考题重点

考试新题

Cloud spanner 优化查询，查询会导致 CPU升高，服务器性能下降。 Provision enough Cloud Spanner nodes to keep high priority total CPU utilization under 65%

  1. 创建一个报警，CPU 45%，重新查询
  2. 创建一个报警，CPU 45%，不重新查询
  3. 创建一个报警，CPU 65%，重新查询
  4. 创建一个报警，CPU 65%，不重新查询 注意要优化，选 D 比较长的

acid 关系型数据库。选 cloud sql 动态路由 BGP

已有 Cloud Storage ，优化花费 setStorageClass。答案好像是设置对象生命周期和修改默认类别，选修改默认类别

生成一个 peer key，上传 public key

有个市场团队的题，要给市场团队建个新项目，答案记不清了，给项目组织加权限，别给市场团队加权限，千万记得选和销售团队的项目id不同的那个项目id

有个机器类型的题，给图片选计算优化型，给标准选普通型

有个找错误的题，好像选的是去 monitoring log 里面看
