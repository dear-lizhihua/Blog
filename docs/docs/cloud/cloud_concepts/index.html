云的概念

参考链接#

  • 什么是云计算？
  • 什么是边缘计算？
  • 云、CDN、IDC 三个概念的区别是什么？有什么相互包含和影响

诸多厂商的回答都喜欢夹带私货，我们需要抽丝剥茧理清概念，避免忽悠和误导。

什么是云(cloud)和云计算(cloud computing)？#

Red Hat 定义的云计算#

一种能够跨网络、按需提供基础架构、服务、平台和应用的交付方式，正在快速取代原本通过硬布线连接进行资源共享的方式。

云计算是指在云中运行工作负载，而云是一种能够抽象、汇集和共享整个网络中的可扩展资源的 IT 环境。云计算和云本身都不属于技术的范畴。

总体来说

  • 云计算是指在云中运行工作负载的功能。
  • 云是一种环境，是运行应用的地方。
  • 技术则是指用于构建和使用云的软件和硬件。

什么是公共云、私有云、混合云、多云？#

  • 云计算服务类型有哪些？公有云，私有云，混合云的区别

  • 公共云：一种利用非最终用户所有的资源创建的云环境，可重新分发给其他租户。

  • 私有云：可广义地定义为：一种专为最终用户而创建，而且通常位于用户的防火墙内（有时也是内部部署）的云环境。

  • 混合云：一种具有一定程度的工作负载可移植性以及编排和管理能力的多云环境。

  • 多云：一个含有多个云环境（公共云或私有云）的 IT 系统，云与云之间可能联网也可能不联网。

云计算服务分类（按照对数据的操作为维度）#

  • 基础架构即服务（IaaS）
  • 平台即服务（PaaS）
  • 软件即服务（SaaS）

云计算服务分类（按照对数据的操作为维度）#

  • 以数据存储为主的存储型云服务
  • 以数据处理为主的计算型云服务
  • 计算和数据存储处理兼顾的综合云服务

什么是边缘计算(edge computing)？#

Red Hat 定义的边缘计算#

云计算使许多组织都将其服务集中在大型数据中心内。但是，新型最终用户体验（例如物联网）要求在靠近网络的外部"边缘"处，也就是物理设备所在的位置提供服务。

边缘计算可以作为混合计算模型的补充。

简单理解#

这是把计算能力放在边缘设备之上，例如我们家里有很多小米的设备，设备内部常都有计算芯片，很多数据没必要直接上传到云上然后在云端计算，而是直接通过本地的芯片计算，再把计算结果存储在本地和上传到云。

IDC#

UCloud 定义的 IDC#

IDC(Internet Data Center)，即互联网数据中心，简称 IDC 机房

IDC 是存放服务器的地方，也是实体服务器的集群。这是随着互联网发展而兴起的服务器托管、租用、运维以及网络接入服务的业务。

通过 IDC 服务，企业或政府单位无需再建立自己的专用机房、铺设昂贵的通信线路，也无需聘请网络工程师，即可解决互联网应用的许多专业需求。

通俗的讲，就是机房+服务。

IDC 与云的关系#

UCloud 定义的关系#

  • IDC 是基础网络资源，而云计算整合、优化了基础网络资源，是纯物理硬件服务器的服务
  • 云是基于 IDC 服务，附加更多的综合服务

计算专门化#

这里采用 UCloud 的回答#

一台服务器，无论是实体还是虚拟的，资源处理能力，或者说运算能力是有限的，当需要处理大量数据，进行大量运算的时候，就会占用系统大量资源，系统其他进程就会处理很慢，体现在服务器就很吃力。这时候有两种选择

  1. 提升服务器性能，比如升级 CPU、提升内存等等
  2. 将耗费计算资源的部分单独划出来，部署在专门进行该服务的服务器上，不占用原服务器上其他进程的资源，例如数据库有专门的云数据库服务，静态网页和图片有专门的文件存储服务。

服务器的性能始终是有限的，我们不能超越现有物理科学的极致。所以，第二种方式就很有必要。

从核心逻辑上看，通过单一职责，把某些特定计算交给专门的机器或者集群负责。
